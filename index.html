Some N-of-1 Trials for Programming / Source Code

This is just a set of N-of-1 experiments that you can run on your own. Here are the experiments you can run directly online:
  • Indentation - Check out what the effect of indentation is.

You can access the code for each experiment under the public git repo: https://github.com/shanenbe/Experiments

In case you want to know more about N-of-1 trials in software science, check out this pub:

  • Hanenberg, Mehlhorn, "Two N-of-1 self-trials on readability differences between anonymous inner classes (AICs) and lambda expressions (LEs) on Java code snippets", Empir. Softw. Eng. 27(2): 33 (2022)
